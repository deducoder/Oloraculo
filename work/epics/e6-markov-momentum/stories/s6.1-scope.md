# Scope: S6.1: Markov Predictor Foundation

## In Scope
- MarkovMomentumPredictor class implementing IPredictor.
- Extracting last 10-15 matches for home and away teams.
- Building transition probability matrix from outcomes (Win, Draw, Loss).
- Computing next state probability based on current state (outcome of the last match).
- Insufficient history check (minimum 5 matches) fallback.
- Complete unit test coverage for the matrix parsing and prediction calculation.

## Out of Scope
- Integration with expected goals models (Poisson/xG) — deferred to S6.2.
- Final selector Momentum Bias logic — deferred to S6.3.

## Done when
- MarkovMomentumPredictor implements IPredictor.
- Transition matrix calculations are tested and verified.
- Unit tests verify fallback behavior.
- Code compiles and passes all dotnet tests.
