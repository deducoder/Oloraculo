# Scope: S6.2: Inertia xG Modifier

## In Scope
- Integrate `MarkovMomentumPredictor` predictions into `GoalPlusRecentContextModel.cs`.
- Calculate team-specific momentum multiplier between `0.90` (-10%) and `1.10` (+10%) using transition probabilities.
- Apply multipliers to expected goals (`homeGoals` and `awayGoals`) in `GoalPlusRecentContextModel.cs`.
- Ensure fallback to `1.0` (no modification) when Markov prediction is degraded.
- Update `Drivers`, `FeaturesUsed`, and `Explanation` in `MatchPrediction` to reflect the applied momentum adjustments.
- Unit test coverage in `Oloraculo.Web.Tests` to verify multiplier calculations and adjustments.

## Out of Scope
- Modifying the core `GoalModel.cs` implementation.
- Implementing momentum bias inside the final selection logic (deferred to S6.3).

## Done when
- `GoalPlusRecentContextModel` adjusts expected goals by up to +-10% using `MarkovMomentumPredictor` outputs.
- Transition-based adjustments fall back to `1.0` if the Markov prediction is degraded.
- Match prediction outputs include features used and details of momentum adjustments in the explanation/drivers.
- Code compiles and passes all dotnet tests.
